# Политика конфиденциальности — Claude → Figma

_Обновлено: 2026._

Плагин «Claude → Figma» импортирует HTML/дизайн в Figma как редактируемые слои.

## Какие данные обрабатываются

- **Ручной режим (вставка HTML):** вся обработка идёт **локально** в плагине (в
  изолированном iframe внутри Figma). HTML и результат **никуда не отправляются**
  и нигде не сохраняются.
- **Режим Claude (по желанию):** если ты подключаешь свой аккаунт Claude через
  MCP-коннектор, HTML, который ты импортируешь, передаётся через relay-сервер
  только для доставки в твой открытый плагин. Данные **не сохраняются** на сервере,
  существуют в памяти лишь на время одной операции и маршрутизируются по приватному
  коду комнаты.

## Что НЕ собирается
- Не собираются персональные данные, аналитика, телеметрия.
- Содержимое твоих файлов Figma не читается и не передаётся.
- Нет рекламы и трекеров.

## Изображения
В импортируемом HTML могут быть ссылки на картинки — плагин загружает их напрямую
с указанных адресов, чтобы поместить в дизайн. Эти запросы идут к сторонним
серверам, указанным в твоём HTML.
