# Облачный relay на своём VPS (оплата картой РФ)

Если зарубежные хостинги (Fly/Render) не принимают карту — арендуй VPS у российского
провайдера и поставь relay туда. Сервер нужен крошечный.

## 1. Арендовать VPS
Провайдеры с картами РФ: **Timeweb Cloud**, **VDSina**, **Aeza**, **reg.ru**, **Selectel**.
- ОС: **Ubuntu 22.04 или 24.04**.
- Тариф: самый младший (1 CPU / 1 ГБ ОЗУ — с запасом; relay ест ~50–100 МБ).
- После создания получишь **IP-адрес** и пароль root (или SSH-ключ).

## 2. Домен (нужен для HTTPS)
claude.ai требует HTTPS, плагин — WSS. Нужен домен, направленный на IP сервера.
- **Бесплатно:** https://www.duckdns.org → войти → создать поддомен (напр. `c2f.duckdns.org`)
  → вписать IP сервера. Готово.
- **Платно:** купить `.ru`-домен (reg.ru/timeweb) и сделать **A-запись** на IP сервера.

Проверь, что домен резолвится в IP (может занять несколько минут).

## 3. Установка одной командой
Подключись к серверу по SSH (`ssh root@IP`) и выполни (подставь свой домен):

```bash
export DOMAIN=c2f.duckdns.org
curl -fsSL https://raw.githubusercontent.com/fggtf24-cyber/claude-to-figma/main/deploy/setup-vps.sh | bash
```

Скрипт сам поставит Node, код, systemd-сервис и Caddy с авто-HTTPS (Let's Encrypt).
По завершении он напечатает адреса.

## 4. Проверка
- В браузере: `https://ТВОЙ-ДОМЕН/health` → должно вернуть **ok**.
- Скажи мне домен — прогоню MCP-хендшейк насквозь.

## 5. Подключение
- **Плагин** (поле WebSocket): `wss://ТВОЙ-ДОМЕН/plugin?room=КОД`
- **claude.ai → Connectors**: `https://ТВОЙ-ДОМЕН/КОД/mcp`
- `КОД` — любой свой длинный секрет, одинаковый в обоих местах (это «комната»).

## Управление
```bash
systemctl status c2f-relay          # статус
journalctl -u c2f-relay -f          # логи
cd /opt/claude-to-figma && git pull && cd server && npm ci --omit=dev && systemctl restart c2f-relay   # обновить
```

## Безопасность
- На сервере нет секретов: комнаты живут только в памяти, токенов в коде нет.
- Открой в фаерволе только 80/443 (Caddy) и 22 (SSH). Порт 8080 (relay) наружу не нужен —
  его проксирует Caddy локально.
- `КОД` — это пароль к твоей комнате: кто знает URL коннектора, тот шлёт в твою Figma.
